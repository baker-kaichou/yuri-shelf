# Yuri Shelf

A static reading-list site for Yuri / Girls' Love manga and literature.

## Files

- `index.html` is the public shelf.
- `admin.html` is the local management panel.
- `style.css` contains shared/public styling.
- `admin.css` contains admin-only styling.
- `manga-data.json` is the portable shelf data file.

## Local Use

Open `index.html` to browse the public shelf. Open `admin.html` to manage entries locally in your browser. The admin panel saves changes to `localStorage`; use **Export JSON** to download an updated `manga-data.json` before publishing.

## Publishing

This project is ready for GitHub Pages because it uses plain HTML, CSS, and JSON with no build step.
